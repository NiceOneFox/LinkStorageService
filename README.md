# Запуск LinkStorageService через Docker Compose

## Быстрый запуск

```bash
# Клонировать репозиторий

# Запустить все сервисы
docker-compose up -d

# Проверить статус
docker-compose ps

# Просмотреть логи
docker-compose logs -f

Импортировать коллекцию postman из файла LinkStorageService API.postman_collection.json
